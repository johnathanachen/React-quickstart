# React Quickstart

Start your project quickly with this template

# Installation

To install the required modules, run the command

```
$ npm i
```

To start up your local dev envirement run:

```
$ npm run dev
```

Run Test

```
$ npm t
```

Serve up docs

```
$ npm run dev
```

# Dependencies

- express
- mongoose
- passport
- bootstrap
- prettier
- blueprintjs
